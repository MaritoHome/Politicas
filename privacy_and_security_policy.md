# Privacy and Security Policy

**Effective Date:** June 23, 2026

This Privacy and Security Policy explains how this application handles user information.

## No Data Collection

This application does not collect, record, access, store, process, transmit, sell, share, or otherwise use any personal or non-personal information from the user's device or account.

The application does not collect:

- Names, email addresses, phone numbers, or contact information.
- Account information or login credentials.
- Device identifiers, advertising identifiers, or IP addresses.
- Location data.
- Photos, videos, audio, files, documents, or contacts.
- Usage data, analytics, diagnostics, crash reports, or activity history.
- Financial, payment, health, biometric, or sensitive personal information.
- Any other information stored on the user's device or associated with the user's account.

## No User Accounts

The application does not require users to create an account, sign in, or provide personal information.

## No Tracking or Analytics

The application does not use tracking technologies, analytics services, advertising services, cookies, profiling systems, or behavioral monitoring tools.

## No Data Sharing

Because the application does not collect any user information, it does not sell, rent, disclose, transfer, or share user information with third parties.

## Device Permissions

The application does not request access to device information or permissions unless they are strictly necessary for a feature explicitly initiated by the user. Any permitted access is processed locally on the device and is never collected, stored, or transmitted by the application.

## Data Security

No user data is stored on external servers or databases. Since the application does not collect or retain user information, there is no user information available for unauthorized access, sale, disclosure, or misuse.

## Children's Privacy

The application does not knowingly collect information from children or from any other user. No personal information is collected regardless of the user's age.

## Third-Party Platforms

The application may be downloaded through third-party platforms such as Google Play. Those platforms may independently process information under their own privacy policies. This application does not receive, control, or store that information.

## Changes to This Policy

If the application's data practices ever change, this policy will be updated before any new data collection or processing begins. The application will not begin collecting user information without providing clear notice and obtaining any consent required by applicable law.
